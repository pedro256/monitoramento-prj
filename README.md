# Sistema de Telemetria
Sistem que permite ter uma forma de visualização geral de dispositivos de uma industria podendo ter visualização e manutenção desses dispositivos, uma tela de observação de funcionamento desses mesmos.

## Tecnologia
- Nextjs 16, tailwind, Next-auth
- Dotnet,Supabase,Docker

## Req. Funcionais:
- Criar Organizações e Dispositivos e Manter os dados dos mesmos.
- ter uma tela de visualização geral desses dispositivos de acordo com a organização
- Permitir ver detalhameento desses dispositivos

## Req. Não funcionais
- Sistema deve permitir Autenticação de usuários
- Sistema de usar para comunicação dos dispositivos o protocolo MQTT, alinhado com MQTT Broker
- Sistema deve estruturar esses dados em um banco POSGRESQL
